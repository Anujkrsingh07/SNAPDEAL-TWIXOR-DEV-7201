# Snapdeal Clone Website

A responsive e-commerce website inspired by Snapdeal, built using React JS and Vite.

## Features

- Home page with modern UI
- Product listing section
- Search functionality
- Add to Cart system
- Buy Now button
- Responsive design
- Fast loading with Vite
- Carousel banner slider

## Technologies Used

- React JS
- Vite
- JavaScript
- CSS
- HTML

## Installation

1. Clone repository

git clone <your-github-link>

2. Open folder

cd project-name

3. Install dependencies

npm install

4. Run project

npm run dev
